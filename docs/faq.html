Questions and answers

Why am I even making this fangame.
For me, this is a way to test the effectiveness of a number of complex decisions in the code.
I follow the principle "Anything can be made from anything", that from the resources of one game it is possible to make another, which, if possible, will retain the features of the first.
Plans for game development.
I plan to make up to 3 game modes:
  1. Missions
  2. Boss fight
  3. The third one will remain a secret for now since its development is still in question.
How to shoot?
Aim with RMB, then shoot with LMB.
What this icon mean?
This icon means that the student can use cover by F.
What this icon mean?
This icon means that the student can jump over some small obstacles using SPACE.
Adding new students.
There are no plans to add new students for the following reasons:
  • Students with the main types of weapons have already been added to the game and therefore adding new students will not change anything in the gameplay (it will only add headaches if skills are added).
  • Students with Railgun and Rocket Launcher will currently have more difficult gameplay than Haruka, so I postponed their addition for the future.
  • Students with Bolt-action Rifle (such as Ui, Hasumi, Koharu, etc.) are currently impossible to balance. I think this will be possible in the future, but not now.
  • Hibiki (Mortar Launcher)... I don't see a way to add her.
Shooting and reloading on the move.
At the current stage it is not planned, in the future I don't know.
Firstly, in the original game, characters shoot and reload while standing still.
Secondly, in the original game there are no animations for this, and creating animations through code is not possible at the moment.
Balancing and improving the gaming experience will be done through other methods.
About adding skills and EX skills.
I would like to add them and initially there were such plans, but everything turned out to be not so simple.
In the original game, the skills are designed for the camera above the battlefield. With the camera from behind the back, it will be quite problematic to make normal aiming for AOE attacks.
Some skills are simply an enhanced shot that will be extremely ineffective against ordinary enemies, it will work against bosses, but there are none now.
I could go on and on, but these are the main reasons why there are no skills.
Even if they are added to the game, it is likely that they will differ from the original.
Can I play on Android?
No, PC only. The main reason is the problem of adapting the controls to the touchscreen. I don't see a way to make the game playable on a phone without fundamentally changing the basic principles of the game.

If you think you know how to do it, you are just imagining it.
The game is difficult.
Until you understand the character's tactics, it really seems like that.
I'm not going to make the game easier on purpose by adding HP or defense to the characters, but after adding some new mechanics it will gradually become easier.
Perhaps in the future it will even have to be made more difficult, at least Haruka will definitely need to be nerfed.
